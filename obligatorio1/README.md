
En esta carpeta (obligatorio1) se encuentra todo el contenido relacionado con la resolución del ejercicio Obligatorio 1, correspondiente a la simulación del Sistema Solar a partir del algoritmo de Verlet.

Cálculo de datos:

Código fuente y Notebook: Se incluyen dos documentos con el código usado para calcular los datos de la simulación. Uno contiene el código en bruto (raw) y el otro es un notebook de Jupyter con explicaciones detalladas.

Archivos generados: El código anterior genera el archivo principal de la simulación (planets_data.dat) y un archivo secundario (energia_momento).

Conservación de la Energía y el Momento:

El archivo energia_momento se utiliza para verificar que ambas magnitudes se conservan durante la simulación.

conservación_energia_grafica: Código utilizado para generar la gráfica (conservación_E_L), la cual ilustra visualmente esta conservación.

Simulación Visual:

animación_planetas: Código original de la simulación visual aportado por la profesora.

animacion_planetas_colores: Adaptación del código anterior que modifica el tamaño de los planetas y les añade color (desarrollado con la asistencia de la IA Gemini).
